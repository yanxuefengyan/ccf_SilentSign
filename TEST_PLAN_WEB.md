# Web端测试计划

## 1. 手语转语音测试
**测试步骤**：
1. 打开Web端应用
2. 切换到"手语转语音"模式
3. 授予摄像头权限
4. 开始摄像头捕获
5. 在摄像头前做出"你好"手势
6. 观察识别结果和语音播放

**预期结果**：
- 识别结果正确显示"你好"
- 语音播放正常
- 置信度显示合理

## 2. 语音转手语测试
**测试步骤**：
1. 切换到"语音转手语"模式
2. 点击"开始录音"按钮
3. 说出"我想挂号"
4. 停止录音
5. 观察手语动画显示

**预期结果**：
- 识别结果正确显示"我想挂号"
- 手语动画正确显示
- 情感表情同步更新

## 3. 离线模式测试
**测试步骤**：
1. 断开网络连接
2. 进行语音转手语操作
3. 观察系统行为

**预期结果**：
- 系统自动切换到离线模式
- 显示离线模式提示
- 正确识别并显示手语动画
